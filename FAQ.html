Foire Aux Questions

Comment créer un compte ?▼
Cliquez sur "Inscription" et remplissez les informations demandées.
J’ai oublié mon mot de passe, que faire ?▼
Utilisez le lien "Mot de passe oublié ?" sur la page de connexion.
Puis-je modifier mon adresse e-mail ?▼
Oui, depuis la page "Profil".
Comment contacter l’équipe ?▼
Envoyez-nous un mail à contact@gusteaus-restaurant.com
Qui a accès aux données capteurs ?▼
Les utilisateurs authentifiés avec les droits appropriés.
Comment supprimer mon compte ?▼
Contactez l’administration pour faire la demande.
Le site est-il sécurisé ?▼
Oui, les mots de passe sont chiffrés et les connexions protégées.
Puis-je utiliser le site sur mobile ?▼
Oui, le site est responsive.
Puis-je suggérer des améliorations ?▼
Bien sûr ! Contactez-nous via e-mail.
Les données sont-elles partagées ?▼
Non, vos données ne sont jamais vendues ni partagées sans votre accord.
Puis-je accéder aux anciennes mesures ?▼
Oui, l’historique est visible selon votre rôle utilisateur.
Les capteurs sont-ils mis à jour en temps réel ?▼
Oui, les données sont rafraîchies automatiquement.
Comment ajouter un capteur ?▼
Seuls les administrateurs peuvent ajouter un capteur depuis leur tableau de bord.
Puis-je exporter les données ?▼
Pas encore, cette fonctionnalité est en développement.
Quelles technologies sont utilisées ?▼
PHP, MySQL, HTML5, JavaScript, CSS3 et Chart.js.
Puis-je partager un accès ?▼
Non, chaque utilisateur doit avoir son propre compte.
Où sont stockées les données ?▼
Les données sont hébergées sur les serveurs sécurisés d’AlwaysData.
Y a-t-il une limite d’utilisation ?▼
Non, vous pouvez utiliser le service autant que nécessaire.
Puis-je recevoir une alerte par mail ?▼
Pas encore, mais une notification est prévue prochainement.
Comment signaler un bug ?▼
Envoyez un e-mail à l’équipe technique via contact@gusteaus-restaurant.com
